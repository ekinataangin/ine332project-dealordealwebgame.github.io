# 🎰 INE 332: Deal or No Deal — Banker Strategy Simulation

A web-based simulation of the **Deal or No Deal** game show, built as a term project for **INE 332: Systems Simulation**.

## 🎮 Play Now

👉 **[Play the Game](https://ekinataangin.github.io/ine332project-dealordealwebgame.io/)**

## 📋 Overview

This project simulates the classic "Deal or No Deal" game show with multiple **banker offer strategies** powered by Monte Carlo simulation techniques. Players can experience the game while observing how different banker policies affect offer amounts.

## 🏦 Banker Policies

| Policy | Description |
|--------|-------------|
| **Baseline (0.8×)** | Offers 80% of the expected value |
| **Risk-Adjusted** | Considers variance/standard deviation in pricing |
| **Dynamic** | Scales offer percentage based on game progress |
| **Adaptive** | Uses context-aware 80% expected value calculation |
